# JavaScript Mini Projects
This repository contains some beginner-friendly JavaScript projects created to practice DOM manipulation, event handling, and basic JavaScript logic.

## 🚀 Projects

### 1️⃣ Background Color Changer
A simple project where the background color of the webpage changes when different buttons are clicked.

**Features**
- Change background color dynamically
- DOM manipulation using JavaScript
- Event listeners for button clicks

---

### 2️⃣ BMI Calculator
This project calculates the Body Mass Index (BMI) based on user input (height and weight).

**Features**
- Takes height and weight as input
- Calculates BMI instantly
- Displays BMI category (Underweight, Normal, Overweight)

---

### 3️⃣ Digital Clock
A real-time digital clock built using JavaScript.

**Features**
- Displays current time
- Updates every second
- Uses JavaScript Date object

---

### 4️⃣ Guess The Number Game
A number guessing game where the user tries to guess a random number between 1 and 100.

**Features**
- Random number generation
- Input validation
- Tracks previous guesses
- Shows hints (Too High / Too Low)
- Limited number of attempts

---
